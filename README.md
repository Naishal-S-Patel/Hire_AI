# 🚀 Hire-AI

### AI Powered Recruitment Automation Platform

Hire-AI is an intelligent recruitment platform designed to automate resume screening, candidate matching, and hiring insights using **Artificial Intelligence and Machine Learning**.

The platform helps recruiters identify the best candidates faster by automatically analyzing resumes, extracting skills, and comparing them with job requirements.

---

# 📌 Problem Statement

Recruiters often spend hours manually reviewing resumes and shortlisting candidates.
With hundreds or thousands of applicants, this process becomes slow, inefficient, and error-prone.

Companies need an automated system that can quickly analyze candidate profiles and help recruiters make better hiring decisions.

---

# 💡 Our Solution

Hire-AI automates the recruitment pipeline using AI and Machine Learning.

The platform:

* Parses resumes using NLP
* Extracts candidate skills and experience
* Matches candidates with job requirements
* Generates AI summaries of candidate profiles
* Provides ATS scoring and fraud detection

This helps recruiters shortlist candidates faster and improve hiring efficiency.

# ⚙️ How to Run Locally

Clone the repository

```
git clone https://github.com/Naishal-S-Patel/Hire-AI
```

Navigate to frontend

```
cd "AI-Based Recruitment"
```

Install dependencies

```
npm install
```

Run the development server

```
npm run dev
```

Open in browser

```
http://localhost:5173
```
